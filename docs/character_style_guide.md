# 角色风格规范

## 固定视觉设定

`artwork/hatch_run/references/canonical-base.png` 是当前审核通过的正式身份与画风基准。标准动作行 0–8 已在 `artwork/hatch_run/decoded/` 完成并通过标准联系表视觉 QA；`running-left` 曾因槽位边界碎片失败，正常重生成后已修复。v2 观察方向仍在制作中。

- 日系二维动画、Q 版但不过度幼化，约 3 头身。
- 长直黑发；偏中分、轻薄碎刘海。头发外轮廓和刘海分区跨帧不变。
- 柔和杏仁眼，小巧自然唇形；五官清楚但不过度放大。
- 红白波点蝴蝶结洛丽塔裙，同款大蝴蝶结发饰，白色蕾丝长靴。
- 全身入镜，底部中心锚点；适合缩小到桌面显示。

默认服装不得混入参考照片中的帽子、电竞耳机、毕业服、灰衣或粉色礼服。不得在动作间随机增加首饰、道具或改变发色。

## 色板与造型

当前色彩以审核通过的 canonical base 为准：头发使用中性近黑并保留少量冷灰高光；服装以清晰红、暖白为主；肤色保持自然柔和；线稿使用深棕黑而非纯黑硬边。后续如生成独立 `character_palette.png`，必须从 canonical base 取样，禁止用参考照片的光照色偏当作固有发色或肤色。

大形优先级：发型轮廓 > 头部蝴蝶结 > 红白裙装轮廓 > 白色长靴 > 小型蕾丝细节。波点在小尺寸下适当减少数量并固定分布逻辑，避免逐帧闪烁。

## 线条、上色与阴影

- 清晰、闭合、稳定的动画线稿；外轮廓略粗，五官和服装内部线略细。
- 平涂赛璐璐上色，一层主阴影和少量硬边高光；光源固定为左上前方。
- 不使用写实皮肤纹理、3D 塑料质感、厚重油画笔触、柔焦光晕或复杂环境反光。
- 透明边缘干净，不得有白边、黑底残留、半透明噪点或内部意外透明裂缝。

## 动画规范

- 画布统一 768×768 RGBA；文件按 `001.png` 连续编号；角色脚底默认对齐底部中心锚点。
- idle 只做呼吸、眨眼和轻微发丝摆动；walk 使用完整步态并可水平镜像；sit 有站到坐的过渡；sleep 使用闭眼呼吸；wave 通过手臂动作表达；happy 可小幅跳起；surprised 短暂后仰；dragged 双脚离地；fall 包含落下、轻微回弹和站稳。
- 发型、蝴蝶结、裙摆和长靴必须随身体有一致的次级运动，不能漂浮或跨帧变形。
- 不使用背景、地面阴影、文字、水印、界面、速度线或与角色分离的装饰特效。

## 禁止改变

发长与刘海分区、眼型、自然唇形、约 3 头身、红白波点裙、头部大蝴蝶结、白色蕾丝长靴不可改变。人物核心特征不能因动作、镜像或生成批次发生年龄感、脸型、发色、服装和配饰漂移。
