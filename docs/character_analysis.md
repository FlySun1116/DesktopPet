## 可见特征与动漫化取舍

稳定锚点包括长直黑发、偏中分的轻薄碎刘海、柔和杏仁形眼睛和小巧自然的唇形。Q 版化时应简化鼻部与面部细节，但保留发际/刘海分区、眼型方向和自然嘴形；避免极端大眼、过度幼化或把滤镜效果固化为五官。

第一版固定服装为红白波点蝴蝶结洛丽塔裙、同款大蝴蝶结发饰和白色蕾丝长靴。这组高对比形状在小尺寸下最适合作为识别锚点。褶边、蕾丝可归纳为少量清晰层次，不能简化掉胸前/头部蝴蝶结、红白配色和靴型。

照片中的帽子、电竞耳机、毕业服、灰色日常衣和粉色礼服属于不同场景元素，不应混入默认角色，也不能在不同动作帧中随机出现。动作制作中最容易漂移的部分是刘海分区、头发长度、蝴蝶结大小与位置、裙摆层数、袖口和长靴高度。

## 一致性锚点

1. 长直黑发轮廓与长度固定。
2. 偏中分、轻薄碎刘海的分区固定。
3. 柔和杏仁眼方向与间距固定。
4. 小巧自然唇形，不使用夸张厚唇或大嘴。
5. 红白波点蝴蝶结洛丽塔裙固定。
6. 同款大蝴蝶结发饰的位置、比例与配色固定。
7. 白色蕾丝长靴的高度和轮廓固定。
8. 全身约 3 头身，头、躯干、腿部关系固定。

## 当前结论与限制

以上内容足以指导第一轮角色设定，但不等于人物本人已认可相似度。正式动作生产前必须先审核正面、45°、侧面、背面和表情设定。任何尚未经过多图一致性及授权复核的生成图，都只能标为草稿或占位素材。
